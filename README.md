# Shopify Custom GTM Pixel for Customer Events

## Installation

1. navigate to [admin.shopify.com](https://admin.shopify.com)
2. navigate to "Customer events" in your store
3. click "Add custom pixel"
4. name it
5. set "Permission" to "Not required"
6. set "Data sale" to "Data collected does not qualify as data sale"
7. copy the code from `custom-pixel.js`
8. paste the code in "Code"
9. update the configuration
10. click "save" at the top of the page
11. click "Connect"
